⚡ Features

主要機能一覧

01
🎲
ドラムロール式ルーレット
8種のミッションからランダム選出。requestAnimationFrameで60fps滑らか動作。easeOutBackのバウンドアニメ付き。チェック済みは除外。
ミッションタブ
02
🌟
やりたいことエリア
忍者修行🥷・ゲーム🎮・読書📖はミッションドラムから独立。カード下部に専用エリア。タップでチェック＆コンフェッティ演出。
ミッションタブ
03
🤫
かくしミッション
ミッション3回以上チェック後に25%の確率で出現。クリアするとゲームが即解放。専用エフェクト＆豪華サウンド付き。
シークレット
04
✅
習慣チェック（39項目）
朝・学校準備・下校後の3セクション。アコーディオン開閉、一括チェックボタン、サブセクション対応。
習慣タブ
05
📊
リアルタイム進捗バー
ヘッダーにstickyで固定。現在タブに合わせて内容が切替。全完了でミント色に変化。PCでは2本並列表示。
全タブ
06
🔒
ゲーム解放システム
ミッションを半分以上クリアorかくしミッションクリアで解放。解放時は豪華オーバーレイ演出。当日限りリセット。
ゲームタブ
07
🔊
Web Audio API効果音
外部音声ファイル不要。オシレーター合成で全効果音を自前生成。ルーレット・チェック・クリア・ゲームで異なる音。
全体
08
📱
PWA対応
manifest.jsonでホーム画面追加可能。apple-touch-icon対応。テーマカラー設定でステータスバーまで統一。
インフラ
